# LinguaNeural Kids - Language Learning App

A Duolingo-like language learning app for children with emotion-aware AI.

## Features
✅ Complete user flow (Welcome → Login → Onboarding → Home)  
✅ English lessons with multiple exercise types  
✅ Voice recognition for speaking exercises  
✅ Audio playback for listening exercises  
✅ State management with Provider  
✅ User progress tracking  
✅ Beautiful UI with Google Fonts  
✅ Responsive design for web/mobile  

## Tech Stack
- **Flutter** - Frontend framework
- **Provider** - State management  
- **Speech to Text** - Voice recognition
- **Just Audio** - Audio playback
- **Google Fonts** - Typography
